01 · about

A polyglot programmer, driven by the challenge rather than the toolchain.

My background is built from a lot of technologies, applied to web projects that mostly demanded serious scalability and lived inside distributed architectures.

I work mainly on web and backend systems, where Ruby and Java are my deepest specialities — but I consider myself language-agnostic. Over the years that's meant Kotlin and Jetpack Compose for Android, Go with ImageMagick for image services, Node.js for real-time websocket layers, and Perl and Shell wherever a pipeline needed gluing together.

I've owned architecture at companies like Nubank and Scott Logic, taught Java and web development as an instructor at Caelum, and built the entire backend behind Santos' largest food-delivery service. Whether the work is a 99.99%-uptime payments platform or a video-publishing pipeline serving one of Brazil's most-visited portals, I care about the same things: scalability, resilience, and code other engineers can live with.

Alongside building, I mentor engineers, define engineering standards, and translate business goals into roadmaps teams can actually deliver.
